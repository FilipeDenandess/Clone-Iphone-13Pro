# Clone-Iphone-13Pro
Projeto front-end desenvolvido com **HTML, CSS e JavaScript**, recriando uma página de apresentação do **iPhone 13 Pro**, com foco em visual moderno, troca de cores do produto e layout responsivo. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1} :contentReference[oaicite:2]{index=2}

## Sobre o projeto

Esta é uma landing page inspirada no site da Apple, apresentando o iPhone 13 Pro com:

- menu de navegação no topo
- seção principal com título, descrição e preço
- botão de compra
- imagem do produto
- seletor de cores interativo
- responsividade para dispositivos móveis

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Exibição de informações do produto
- Alteração da imagem do iPhone ao clicar nas opções de cor
- Destaque visual da cor selecionada
- Layout adaptado para telas menores
